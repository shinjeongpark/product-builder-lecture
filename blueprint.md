# Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers.

## Project Outline

### Style and Design
*   **Layout:** A clean and simple layout with a centered content area.
*   **Typography:** Clear and readable fonts.
*   **Color Scheme:** A simple color scheme with a primary color for the button and generated numbers.
*   **Iconography:** A lottery ball icon for the header.

### Features
*   **Lotto Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Web Component:** The lottery number generator is implemented as a reusable Web Component.
*   **Responsive Design:** The layout is responsive and works on different screen sizes.

## Current Plan

*   **1. Create `index.html`:** Set up the basic HTML structure with a `lotto-generator` custom element.
*   **2. Create `main.js`:** Implement the `LottoGenerator` web component, including the number generation logic and UI.
*   **3. Create `style.css`:** Add styles for the page and the `lotto-generator` component.
